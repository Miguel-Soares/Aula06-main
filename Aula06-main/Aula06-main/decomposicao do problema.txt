ENUNCIADO:
Faça um programa em Java para simular uma parte do funcionamento de
uma agenda de contatos. Você deverá ter as opções de incluir uma pessoa
e listar todas as pessoas cadastradas. Cada pessoa terá nome, sobrenome e
telefone de contato. O número máximo de contatos que podem ser salvos é
definido por você.

SUBPROBLEMAS / TAREFAS A RESOLVER
- Incluir uma pessoa / contato
- Listar as pessoas / contatos
- Definir a quantidade limite de pessoas / contatos

NECESSIDADES IDENTIFICADAS
- Incluir uma pessoa / contato
  - Dados do contato (nome, sobrenome, telefone)
    - pedir ao usuário
  - Local para armazenar as pessoas / contatos
  - Saber se o limite de contatos já foi atingido
    - só cadastro quando ainda tenho espaço / capacidade disponível

IMPLEMENTAÇÃO
- Incluir uma pessoa / contato
  - Dados do contato (nome, sobrenome, telefone)
    - Criar uma classe Contato - OK
    - pedir ao usuário - OK
      - Usar Scanner para ler do console - OK
  - Local para armazenar as pessoas / contatos - OK
    - Usar uma coleção (lista, set, etc.), vetor, etc. - OK
  - Saber se o limite de contatos já foi atingido
    - contar a cada contato incluído - OK
    - antes de inserir mais um, perguntar se espaço disponível é maior do que 0 - OK
    - espaço disponível = limite - quantidade de cadastrados
    - só cadastro quando ainda tenho espaço / capacidade disponível
      - se tenho espaço disponível, incluo e aviso - OK
      - se não tenho, informo o usuário sobre o problema - OK
